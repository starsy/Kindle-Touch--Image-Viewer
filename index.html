Kindle-Touch--Image-Viewer

An extension for GUI Launcher on Kindle Touch: Image Viewer

Prerequisite:

  • Kindle Touch w/ Firmware 5.1.0

Installation:

  1. Jailbreak Kindle Touch (http://wiki.mobileread.com/wiki/Kindle_Touch_Hacking#data.tar.gz_method)
  2. Install GUI Launcher (http://wiki.mobileread.com/wiki/Kindle_Touch_Hacking#GUI_Launcher)
  3. Download this extension as a zip/tgz file (https://github.com/starsy/Kindle-Touch--Image-Viewer/zipball/master)
  4. Unpack it into the "extensions" directory on mounted Kindle removable drive
  5. Create an "images" directory on mounted Kindle removable drive

Usage:

  1. Put image files under the "images" directory
  2. "Menu" -> "Launcher" -> "Launch Image Viewer" to view images
